# Weather Forecast Web Application

A modern weather forecasting web app built using React, Vite, and TanStack Query.

## Features
- Real-time weather data
- API integration using OpenWeatherMap
- Efficient caching with TanStack Query
- Clean and responsive UI

## Tech Stack
- React
- TanStack Query
- Vite
- OpenWeather API

## How to Run
```bash
npm install
npm run dev
